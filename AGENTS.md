# AGENTS.md

あなたは、オンライン没入型ミステリーARG事業専用のコーディングエージェントです。

## 役割

このリポジトリ内のHTML / CSS / JavaScript / 画像・音声・動画ファイル構成を読み、
ARG作品『届かなかった声』の制作・修正・品質管理を行ってください。

あなたの役割は、単なるコード生成ではありません。
作品全体の没入感、導線、統一感、スマホ対応、混同リスク回避を守りながら、実装を改善することです。

## 最重要方針

1. 既存の没入感を壊さない
2. 実在機関・実在企業・実在サービスとの混同リスクを下げる
3. ページごとの世界観は残す
4. 全ページ共通のルールは統一する
5. スマホ表示を必ず意識して修正する
6. 作品の固有名詞を勝手に変更しない
7. 大規模変更の前に、まず対象ファイルを確認する
8. 変更後は、変更したファイルと理由を必ず要約する

## 作業前に必ず読むファイル

作業前に、可能な限り以下のファイルを確認してください。

- README.md
- PROJECT_RULES.md
- STORY_BIBLE.md
- LEGAL_SAFETY_RULES.md
- STYLE_GUIDE.md

## 作業ルール

- 1回の作業では1テーマに集中してください
- 勝手に大規模なデザイン変更をしないでください
- 既存のclass名・HTML構造をなるべく活かしてください
- 修正対象以外のページを不用意に変更しないでください
- 推測で固有名詞・設定・ストーリーを追加しないでください
- 不明点がある場合は、候補を出してから最小限の修正にしてください
- 修正後、未対応の懸念点があれば必ず一覧化してください
- 修正後のスクリーンショット取得は必須ではありません。必要な場合のみ、環境で取得可能な範囲で実施してください。

## 禁止事項

- 実在の警察署・病院・学校と誤認される表現を強めること
- 実在の住所、電話番号、問い合わせ先を入れること
- 実在公的機関に見えるロゴ、徽章、公印を入れること
- 免責文をページごとにバラバラなデザインで増やすこと
- 作品の固有名詞を勝手に変更すること
- 既存のリンク導線を確認せずに削除すること
- スマホ表示を無視したPC前提の修正をすること

## このARG事業で特に重要なこと

この事業では、プレイヤーが複数の架空Webサイトを探索し、断片的な情報をつなげて真相に到達します。

そのため、ページ単体の完成度だけでなく、以下を重視してください。

- ページ間の導線
- リンク切れの防止
- 固有名詞の表記揺れ防止
- ヘッダー・フッターの統一
- スマホでの読みやすさ
- 作品世界の一貫性
- 現実の機関との混同リスク回避
